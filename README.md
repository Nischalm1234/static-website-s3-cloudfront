# Static Website Hosting Using AWS
Implemented a scalable static website using S3 and CloudFront with CDN caching and cache invalidation.

## Services Used
- Amazon S3
- CloudFront

## Architecture
User -> CloudFront -> S3

## Features
- Static hosting
- CDN distribution
- Cache invalidation
